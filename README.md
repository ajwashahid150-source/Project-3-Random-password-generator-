# 🔐 Enterprise Random Password Generator

A Python-based application that generates strong, secure, and random passwords using uppercase letters, lowercase letters, digits, and special characters. This project is designed to help users create secure passwords for their online accounts while demonstrating the use of Python's built-in `random` and `string` modules.

---

## 🚀 Features

- Generate strong and secure random passwords
- User-defined password length
- Minimum password length validation (8 characters)
- Includes uppercase and lowercase letters
- Includes numbers (0–9)
- Includes special characters
- Fast and easy to use
- Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Python 3
- Random Module
- String Module

## ▶️ How to Run

1. Clone this repository.

```bash
git clone https://github.com/your-username/Enterprise-Random-Password-Generator.git
```

2. Open the project in Visual Studio Code.

3. Run the program.

```bash
python main.py
```

4. Enter the desired password length (minimum 8).

5. The program will generate a secure random password.
